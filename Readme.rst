===========
Love Letter
===========

An implementation of the card game *Love Letter* for CSC 4444 Artificial
Intelligence with Dr. Jianhua Chen.

Features
--------

+ Automated game logic
+ Playable AI Bots
+ (stretch) GUI interface
+ (stretch) Networked games with **IBM BlueMix**
